# ctd-prework

## This code snippet asks questions that need to be asked before buying an alcoholic drink.
### 3 variable types: string, number and boolean
### Questions needed answered: Conditional for each question.
#### 1. Is this your license- does name match?
#### 2. Are you 21 years old the legal age?
#### 3. Can you afford the drink with the money you have?
#### 4. Personal Question:Could action be harmfull to you are others?

## Instructions to user.
### Pretend Susan asks bartender for drink.  Bartender must determine if it is appropriate to sell her a drink. User must answer the input questions to be answered by either the bartender or Susan. The end result will determine if she gets a drink.
